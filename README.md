# PropIntel AI

AI-powered Real Estate platform integrating Full Stack Development, Generative AI, Agentic AI, and DevOps.

## Modules

### 1. Real Estate Platform
- User authentication
- Property listing
- Add property
- Search and filter

### 2. Generative AI Learning Hub
- RAG-based quiz generation
- FAISS vector database
- Groq LLM integration
- AI explanations
- Score, streak, badges
- AI recommendation engine

### 3. Agentic AI Support Center
- Classifier Agent
- NER Agent
- Response Agent
- Support ticket dashboard
- Ticket status update

## Tech Stack

- React + Vite
- Flask
- SQLite
- FAISS
- Sentence Transformers
- Groq API
- Docker
- AWS EC2
